# teste_coopersystem

<h1>Como Instalar</h1>
<br>
<p>Com o WordPress instalado e tema configurado,</p>
<p> basta fazer o download da pasta 'anuncios' dentro da pasta \wp-content\plugins</p>
<br>
<p> Ative o plugin dentro do painel administrativo do seu WordPress, em seguida você terá acesso ao menu 'Anuncios', dentro do painel </p>
<p> Dentro do plugin você pode criar, alterar ou excluir um anuncio.</p>
<br>
<h2>Como listar os Anúncios</h2>
<br>
<p>Para listar os anúncios basta adicionar o shortcode [list-anuncios] em qualquer página do seu site através do painel administrativo.</p>
